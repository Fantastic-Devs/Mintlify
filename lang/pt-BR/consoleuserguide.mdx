## Painel

### Visualização do Mapa

<img height="250" src="/images/console-screen-shot.png" />

O Painel do Portal do Administrador permite que você visualize dados de pesquisa geograficamente por meio de um mapa – a partir dele, você pode identificar rapidamente quaisquer padrões/problemas que possa ter com seus clientes validando sua localização.

Há quatro seções na tela Visualização do Mapa

- **Mapa**<br />
Esta seção detalha a representação geográfica das pesquisas para o dia atual

- **Atividade**<br />
Esta seção exibe um gráfico de pizza mostrando o número total de pesquisas que ocorreram na data selecionada. As pesquisas feitas dentro da região são representadas por verde e as pesquisas que falham ou estão fora são representadas por vermelho.

- **Dispositivos**<br />
Esta seção exibe um gráfico de pizza mostrando o número total de pesquisas, divididas por tipo de dispositivo, na data especificada.

- **3 principais motivos para falha**<br />
Este gráfico mostra os três principais motivos para pesquisas malsucedidas e os divide pelo motivo pelo qual falharam – Para visualizar todas as falhas, clique no link 3 principais motivos para falha

A visualização de transações exibe todas as pesquisas que correspondem ao filtro definido.

Os dados disponíveis na Visualização de lista são os seguintes:

- **ID da conta** ID da conta do cliente, este é um link e se você clicar nele a página de informações da conta abre detalhando todas as informações sobre essa conta

- **ID do dispositivo** ID do dispositivo, este é um link e se você clicar nele a página de informações do dispositivo abre detalhando todas as informações sobre esse dispositivo

- **Data e hora** Data e hora em que a pesquisa ocorreu

- **Resultado da pesquisa** Isso exibe o resultado da pesquisa – Sucesso ou o motivo da falha da pesquisa.

- **ID da transação** ID de pesquisa exclusivo

- **Latitude e longitude** Detalhes do local

- **Motivo da pesquisa** Isso exibe o motivo da pesquisa (por exemplo, o cliente estava fazendo login ou fazendo uma aposta)

Você pode manipular ainda mais os dados usando os seguintes recursos

- **Colunas** Use este recurso para selecionar quais colunas deseja visualizar. Ao clicar nesta opção, você verá uma lista de todas as colunas com caixas de seleção ao lado delas. Use a caixa de seleção para ativar e desativar a coluna.

- **Filtros** Use o recurso Filtros para filtrar ainda mais os resultados. Ao clicar nesta opção, você verá o seguinte:

Use o seletor Colunas para escolher a coluna que deseja filtrar ainda mais

Use o menu suspenso Operador para selecionar como deseja filtrar a coluna escolhida

Finalmente, digite o valor pelo qual deseja filtrar a coluna escolhida.

- **Densidade** Para alterar a altura das linhas, você pode usar o recurso de densidade:

Use the Export feature to download the data as a Comma-Separated Variables (CSV) file or print them out 

When you hover your mouse over each column header, two buttons appear on it: 

 

- **Ordenar/Classificar**

Se você clicar na seta ao lado do título, ela classificará a data na coluna selecionada – se a seta estiver apontando para baixo, ela classificará os dados em ordem crescente, se a seta estiver apontando para cima, ela classificará os dados em ordem decrescente

- **Menu de recursos**

Se você clicar neste botão, ele abrirá o menu de recursos. Ele contém várias opções que permitem que você manipule os dados nesta coluna.

- **Classificar por ASC** Isso classifica a coluna em ordem crescente

- **Classificar por DESC** Isso classifica a coluna em ordem decrescente

- **Filtrar** Isso abre o recurso de filtro com o nome da coluna já preenchido

- **Ocultar coluna** Isso oculta a coluna selecionada

- **Gerenciar colunas** Isso abre o recurso Colunas e permite que você ative e desative as colunas

## Atualizar (Automático)

Se desejar atualizar/atualizar os dados no Painel, clique nas setas no canto superior direito dos dados, ao lado do último horário de atualização.

Você pode ativar as "Atualizações automáticas" para que os dados estejam sempre atualizados. Esta alternância está no canto inferior direito do painel

Pesquisar

Existem dois tipos de pesquisas disponíveis, Básica e Avançada.

A Pesquisa Básica permite que você pesquise pelo seguinte:

ID da transação

ID da conta

ID do dispositivo

Coordenadas de localização (latitude, longitude)

Endereço IP

A pesquisa avançada também permite que você pesquise pelo seguinte:

Período da transação

Tipo de dispositivo

Clientes localizados por

Motivo da falha

Pesquisa básica

Para usar a pesquisa básica, primeiro selecione o que deseja pesquisar selecionando a opção relevante no menu suspenso a seguir:

Depois de selecionar a opção que deseja pesquisar, use o campo de texto para inserir o que deseja pesquisar.

Pesquisa avançada

A pesquisa avançada permite que você filtre ainda mais a consulta

Período de transação

O Período de transação permite que você pesquise por pesquisas feitas na semana, mês e 3 meses anteriores – ou você pode definir seu próprio intervalo usando o seletor de datas personalizadas

Tipo de dispositivo

A seção Tipo de dispositivo permite que você pesquise por pesquisas feitas por um tipo específico de dispositivo, por exemplo, iOS, Android etc.

Clientes localizados por

A seção Clientes localizados por permite que você pesquise por pesquisas onde a localização foi determinada usando uma tecnologia específica. As opções disponíveis estão listadas abaixo:

Motivo da falha
